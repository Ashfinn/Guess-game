# Guessing Game

Improved version of the original game with minor readability and instruction tweaks.

## Features
- Two difficulty levels: **Hard (5 lives)**, **Easy (10 lives)**
- Interactive, text-based gameplay
- Random number generated between 1 and 100
- Lives system to increase challenge
- Minor improvements in code readability and output formatting

## How to Play
1. Run the game:
```bash
python guessing-game.py
````

2. Choose difficulty level
3. Enter guesses until you guess correctly or run out of lives
